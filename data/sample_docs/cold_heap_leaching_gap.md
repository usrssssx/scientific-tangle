Обзор фиксирует холодный климат и кучное выщелачивание для медных и урановых руд, но подтвержденных экспериментов для комбинации холодный климат + кучное выщелачивание + никелевая руда не найдено. Есть смежные кейсы по управлению температурой штабеля, теплоизоляции и сезонному орошению.

Пробел знаний: нет надежной связки материал-режим-условие для никелевой руды при отрицательных температурах. Требуется пилотный эксперимент с контролем температуры, извлечения, расхода кислоты и выхода металла.
